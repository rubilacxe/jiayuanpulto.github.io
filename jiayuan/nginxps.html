nginx+php+smarty
日期：2015年11月9日

    在搬瓦工的服务器上搭个环境smarty环境。

    首先还是安装环境：

  • nginx安装：https://www.nginx.com/resources/wiki/start/topics/tutorials/install；

  • php安装：yum install php；

  • php-fpm安装：yum install php-fpm；

  • smarty安装：wget https://github.com/smarty-php/smarty/releases/tag/v3.1.24，cp smarty-3.1.24/libs/* /data0/smarty -r；
