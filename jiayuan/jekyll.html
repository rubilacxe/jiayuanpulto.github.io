jekyll试用
日期：2015年11月9日

    在搬瓦工买了个服务器，搭个环境玩。之前一直想折腾个博客，懒人各种没治。。。偶然间看到这个http://yongyuan.name/blog/collect-jekyll-theme.html，有很多不错的模板可以用，对于我这种JS与CSS新手来说还是不错的选择。接下来就是在服务器上安装jekyll（github支持jekyll）。

  • 系统：CentOS release 6.7 (Final)；
  • nginx安装: http://my.oschina.net/VincentJiang/blog/224993；
  • jekyll安装： yum install ruby，yum install rubygems，yum install ruby-devel，gem install jekyll提示版本太低（ruby version>1.9.5）；
  • ruby安装：https://gist.github.com/mustafaturan/8290150；
  • nodejs安装：http://www.jianshu.com/p/783906f10d58；
  • jekyll安装：/usr/local/bin/gem install jekyll；

    jekyll官方文档：http://jekyllcn.com/docs/configuration/，创建项目，配置nginx即可访问博客~~~
